# 📸 AI Photo Generator

An AI-powered web application that transforms standard product photos into professional marketing assets using **Stable Diffusion XL (SDXL)**.

## 🚀 Features
* **Background Synthesis:** Intelligent background replacement for products.
* **Simple Prompting:** Control the scene with 4-5 word English commands.
* **High-Resolution Output:** Generates 1024x1024 HD images.
* **User Authentication:** Secure login with multi-language support.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Backend:** Python
* **AI Model:** SDXL via Hugging Face Inference API
* **Image Processing:** PIL (Pillow)
